# OCM Podinfo
This project is a fork of [podinfo](https://github.com/stefanprodan/podinfo) enriched with ocm functionality. It is used here as example application.
## About this project

Poodinfo is a tiny web application made with Go that showcases best practices of running microservices in Kubernetes. Podinfo is used by CNCF projects like Flux and Flagger for end-to-end testing and workshops.
